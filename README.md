# garyvirk.com

Personal website, hosted with GitHub Pages (branch `main`, root).

## Deploy

```bash
./deploy.sh "optional commit message"
```

Edit files, run the script, changes are live at https://garyvirk.com within ~1 minute.

## Notes

- `index.html` is the homepage; images go in `assets/`.
- `CNAME` binds the custom domain — do not delete.
